## üöÄ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org)
- [npm (Node Package Manager)](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)
- Python (version >= 3.7)

### Installation

Follow these steps to set up and run StudyGeniePro on your system:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/srkeyyyy/StudyGeniePro.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd StudyGeniePro
    ```

3. **Install dependencies:**

    ```bash
    npm install  # or yarn install
    ```

4. **Run the application:**

    ```bash
    npm start
    ```

Open your browser and go to `http://localhost:3000` to access the application.

## üìñ Usage

1. **Login or Sign Up:** Start by signing up or logging in to your account.
2. **Input Subject Details:** Enter the subject or topic you want to create a curriculum for.
3. **Generate Curriculum:** Let StudyGeniePro create a detailed 10-module curriculum tailored to your needs.
4. **Learn and Quiz:** Use the dynamically generated quizzes to master each topic.
